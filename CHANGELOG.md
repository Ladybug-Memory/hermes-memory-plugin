# Changelog

## 1.0.0

- initial standalone release of the Hermes Ladybug Memory directory plugin
- local columnar embedded graph database (*.lbdb) with BM25 search, importance scoring, and graph edges
- eight agent tools: ladybug_store, ladybug_search, ladybug_recall, ladybug_update, ladybug_delete, ladybug_link, ladybug_related, ladybug_entity
- background prefetch, built-in memory mirroring, and compression hook
- optional GLiNER2 entity extraction via ladybug-memory[extract]
